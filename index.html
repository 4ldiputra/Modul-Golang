1. Pengenalan Golang

1.1 Definisi dan Sejarah

Apa Itu Golang?

Go (atau Golang) adalah bahasa pemrograman open-source yang dikembangkan oleh Google pada tahun 2007 dan dirilis ke publik pada tahun 2009. Diciptakan oleh Robert Griesemer, Rob Pike, dan Ken Thompson, Go dirancang untuk mengatasi masalah dalam pengembangan software skala besar.

Keunggulan Go:

  • Kompilasi cepat dan performa tinggi
  • Concurrency yang mudah dengan goroutines
  • Garbage collection otomatis
  • Sintaks yang sederhana dan mudah dipelajari
  • Standard library yang lengkap
  • Cross-platform compilation

1.2 Instalasi Golang

Instalasi di Windows

  1. Kunjungi situs resmi https://go.dev/dl/.
  2. Download installer Go untuk Windows (file .msi).
  3. Jalankan installer dan ikuti instruksi hingga selesai.
  4. Setelah selesai, buka Command Prompt lalu ketik:
    go version
    Jika muncul versi Go, berarti instalasi berhasil.

Instalasi di macOS dengan Homebrew

  1. Pastikan Homebrew sudah terpasang di sistem. Jika belum, install dengan perintah:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install Go menggunakan brew:
    brew install go
  3. Cek instalasi dengan perintah:
    go version
    Jika muncul versi Go, berarti instalasi berhasil.

Program Hello World Pertama

package main import "fmt" func main() { fmt.Println("Hello, World!") }

  • Install Go di komputer Anda
  • Jalankan program Hello World

2.3 Input dan Output

Dalam Go, untuk melakukan input/output biasanya menggunakan package fmt.

package main import "fmt" func main() { var name string var age int fmt.Print("Masukkan nama: ") fmt.Scanln(&name) fmt.Print("Masukkan umur: ") fmt.Scanln(&age) fmt.Println("Halo,", name, "umur Anda", age, "tahun") }

Penjelasan: - fmt.Print() → menampilkan teks tanpa pindah baris. - fmt.Println() → menampilkan teks dengan pindah baris. - fmt.Scanln(&variabel) → membaca input dari user dan menyimpannya ke variabel.

Latihan Praktek

  • SOAL 1:Buatlah program yang menggunakan inputan dan mendeklarasikan variabel untuk data mahasiswa (nama, NIM, IPK, status aktif)

Contoh Penyelesaian

Contoh Program Kalkulator Sederhana

package main import "fmt" func main() { var num1, num2 float64 var operator string fmt.Print("Enter first number: ") fmt.Scan(&num1) fmt.Print("Enter operator (+, -, *, /): ") fmt.Scan(&operator) fmt.Print("Enter second number: ") fmt.Scan(&num2) switch operator { case "+": fmt.Println("Result:", num1+num2) case "-": fmt.Println("Result:", num1-num2) case "*": fmt.Println("Result:", num1*num2) case "/": if num2 != 0 { fmt.Println("Result:", num1/num2) } else { fmt.Println("Error: Division by zero") } default: fmt.Println("Invalid operator") } }

4. Percabangan

4.1 IF, ELSE IF, ELSE

Percabangan digunakan untuk membuat keputusan dalam program berdasarkan kondisi tertentu.

Struktur Dasar Percabangan IF-ELSE

if kondisi { // Kode yang dijalankan jika kondisi true } else { // Kode yang dijalankan jika kondisi false }

Contoh Program Percabangan IF-ELSE

package main import "fmt" func main() { var angka int fmt.Print("Masukkan sebuah angka: ") fmt.Scan(&angka) if angka%2 == 0 { fmt.Println("Angka genap") } else { fmt.Println("Angka ganjil") } }

Latihan Praktek

  • Implementasikan beberapa percabangan dalam program
  • SOAL: Buatlah program dalam bahasa Go yang meminta pengguna memasukkan sebuah bilangan bulat. Program harus menentukan apakah bilangan tersebut positif atau negatif, kemudian menampilkan hasilnya ke layar.

Struktur Dasar Percabangan IF-ELSE IF-ELSE

if kondisi1 { // Kode yang dijalankan jika kondisi1 true } else if kondisi2 { // Kode yang dijalankan jika kondisi2 true } else { // Kode yang dijalankan jika semua kondisi false }

Contoh Program Percabangan IF-ELSE IF-ELSE

package main import "fmt" func main() { var num int fmt.Print("Masukkan angka: ") fmt.Scan(&num) if num > 0 { fmt.Println("Angka positif") } else if num < 0 { fmt.Println("Angka negatif") } else { fmt.Println("Angka nol") } }

Latihan Praktek

  • Implementasikan beberapa percabangan dalam program
  • SOAL: Buat program untuk menentukan kategori umur:
    < 13 → Anak-anak
    13–17 → Remaja
    18–59 → Dewasa
    ≥ 60 → Lansia

Contoh Kode

package main import "fmt" func main() { var umur int fmt.Print("Masukkan umur Anda: ") fmt.Scan(&umur) if umur < 13 { fmt.Println("Kategori: Anak-anak") } else if umur >= 13 && umur <= 17 { fmt.Println("Kategori: Remaja") } else if umur >= 18 && umur <= 59 { fmt.Println("Kategori: Dewasa") } else if umur >= 60 { fmt.Println("Kategori: Lansia") } else { fmt.Println("Umur tidak valid") } }

Latihan Praktek Grade

  • Buat program penilaian sederhana dengan menggunakan percabangan if-else
  • SOAL: Buatlah program yang menentukan grade berdasarkan nilai (A: 90–100, B: 80–89, C: 70–79, D: 60–69, E: <60)

4.2 Nested IF Statement

Nested if adalah percabangan if di dalam if lainnya. Biasanya digunakan untuk memeriksa lebih dari satu kondisi secara berurutan.

package main import "fmt" func main() { ipk := 3.8 penghasilanOrtu := 3000000 if ipk >= 3.5 { if penghasilanOrtu < 5000000 { fmt.Println("Selamat, kamu berhak mendapatkan beasiswa!") } else { fmt.Println("IPK bagus, tapi penghasilan orang tua terlalu tinggi.") } } else { fmt.Println("Maaf, IPK belum memenuhi syarat.") } }

Latihan Praktek

  • SOAL: Buatlah program untuk menentukan status kelulusan siswa berdasarkan nilai dan persentase kehadiran.
  • Jika nilai < 75 → tampilkan "Tidak lulus, nilai kurang dari 75."
  • Jika nilai ≥ 75 dan kehadiran ≥ 80 → tampilkan "Lulus dengan baik."
  • Jika nilai ≥ 75 tetapi kehadiran < 80 → tampilkan "Nilai cukup, tapi kehadiran kurang."

4.3 Switch Case

Switch case digunakan untuk percabangan multi-kondisi yang lebih efisien daripada if-else bertingkat.

Struktur Dasar Switch Case

switch ekspresi { case nilai1: // kode jika ekspresi == nilai1 case nilai2: // kode jika ekspresi == nilai2 default: // kode jika tidak ada yang cocok }

Contoh Switch Case Sederhana

package main import "fmt" func main() { var pilihan int fmt.Print("Pilih menu (1-3): ") fmt.Scan(&pilihan) switch pilihan { case 1: fmt.Println("Anda memilih menu 1") case 2: fmt.Println("Anda memilih menu 2") case 3: fmt.Println("Anda memilih menu 3") default: fmt.Println("Pilihan tidak valid") } }

Contoh Switch Case dengan percabangan IF-ELSE

package main import "fmt" func main() { var menu, nilai int fmt.Print("Pilih mata pelajaran (1=Matematika, 2=Bahasa): ") fmt.Scan(&menu) switch menu { case 1: fmt.Print("Masukkan nilai Matematika: ") fmt.Scan(&nilai) if nilai >= 75 { fmt.Println("Lulus Matematika") } else { fmt.Println("Tidak lulus Matematika") } case 2: fmt.Print("Masukkan nilai Bahasa: ") fmt.Scan(&nilai) if nilai >= 70 { fmt.Println("Lulus Bahasa") } else { fmt.Println("Tidak lulus Bahasa") } default: fmt.Println("Pilihan tidak valid") } }

Latihan Praktek

  • Implementasikan switch case dalam program
  • SOAL: Buatlah program dalam bahasa Go yang menampilkan pilihan buah:

    Jika pengguna memilih Apel, minta input harga. Jika harga > 20000 tampilkan "Apel mahal", selain itu "Apel terjangkau".

5. Perulangan

5.1 For Loop

Di Golang, hanya ada satu jenis loop yaitu for loop. Namun, for loop di Go sangat fleksibel dan bisa digunakan untuk berbagai keperluan perulangan.

Struktur For Loop

for inisialisasi; kondisi; increment/decrement { // Kode yang akan diulang }

Contoh For Loop Ascending (1, 2, 3, 4, 5)

package main import "fmt" func main() { // Mencetak angka 1-5 for i := 1; i <= 5; i++ { fmt.Print(i, " ") } fmt.Println() }

Contoh For Loop Descending (5, 4, 3, 2, 1)

package main import "fmt" func main() { // Mencetak angka 5-1 for i := 5; i >= 1; i-- { fmt.Print(i, " ") } fmt.Println() }

Latihan For Loop dari 1 sampai 100 Ascending

package main import "fmt" func main() { // Ascending (1 sampai 100) fmt.Println("Ascending (1 ke 100):") for i := 1; i <= 100; i++ { fmt.Print(i, " ") } fmt.Println() }

Latihan Praktek

  • Buat program yang menggunakan berbagai jenis perulangan
  • SOAL: Buatlah program Descending (100 ke 1)

5.2 For sebagai While dan Do-While Loop

For Loop sebagai While Loop

package main import "fmt" func main() { j := 1 for j <= 5 { fmt.Print(j, " ") j++ } fmt.Println() }

For Loop sebagai Do-While Loop

package main import "fmt" func main() { k := 1 for { fmt.Print(k, " ") k++ if k > 5 { break } } fmt.Println() }

Contoh penggunaan for dengan switch-case

package main import "fmt" func main() { // Pola bintang segitiga for i := 1; i <= 5; i++ { for j := 1; j <= i; j++ { fmt.Print("* ") } fmt.Println() } }

📊 Array di Golang

6.1 Pengenalan Array

Array adalah struktur data yang digunakan untuk menyimpan kumpulan elemen dengan tipe data yang sama dalam lokasi memori yang berdekatan. Di Go, ukuran array adalah bagian dari tipenya dan bersifat tetap (fixed size).

Perbedaan dengan C: Array di Go di-pass by value (dikopi), sedangkan di C di-pass by reference.

Deklarasi Array

package main

import "fmt"

func main() {
    // Cara 1: Deklarasi tanpa inisialisasi
    var numbers [5]int
    fmt.Println(numbers) // Output: [0 0 0 0 0]

    // Cara 2: Deklarasi dengan inisialisasi
    var fruits [3]string = [3]string{"Apple", "Banana", "Orange"}
    fmt.Println(fruits)

    // Cara 3: Short declaration (lebih umum)
    scores := [5]int{10, 20, 30, 40, 50}
    fmt.Println(scores)

    // Cara 4: Inisialisasi sebagian
    partial := [5]int{1, 2}
    fmt.Println(partial) // Output: [1 2 0 0 0]

    // Cara 5: Compiler menghitung ukuran otomatis
    auto := [...]int{1, 2, 3, 4}
    fmt.Println(auto)
    fmt.Println("Panjang:", len(auto)) // Output: 4
}

Mengakses dan Mengubah Elemen Array

package main

import "fmt"

func main() {
    numbers := [5]int{10, 20, 30, 40, 50}

    // Mengakses elemen (indeks dimulai dari 0)
    fmt.Println("Elemen pertama:", numbers[0])  // 10
    fmt.Println("Elemen terakhir:", numbers[4]) // 50

    // Mengubah nilai elemen
    numbers[2] = 35
    fmt.Println("Setelah diubah:", numbers) // [10 20 35 40 50]

    // Akses di luar batas akan error saat compile
    // fmt.Println(numbers[10]) // ERROR: index out of range
}

Menghitung Panjang Array

package main

func main() {

    fmt.Println("Cara 1: For loop tradisional")

        fmt.Printf("numbers[%d] = %d\n", i, numbers[i])
    }

    // CARA 2: Range - mendapat index dan value
    fmt.Println("\nCara 2: Range dengan index dan value")
    for index, value := range numbers {
        fmt.Printf("[%d] = %d\n", index, value)
    }

    // CARA 3: Range - hanya value (index diabaikan dengan _)
    fmt.Println("\nCara 3: Range hanya value")
    for _, value := range numbers {
        fmt.Print(value, " ")
    }
    fmt.Println()

    // CARA 4: Range - hanya index
    fmt.Println("\nCara 4: Range hanya index")
    for index := range numbers {
        fmt.Print(index, " ")
    }
    fmt.Println()
}

package main

import "fmt"

func main() {
    var nilai [5]int
    var maks, min, total int
    var rata float64

    fmt.Println("=== Program Nilai Siswa ===\n")

    // Input nilai siswa
    for i := 0; i < len(nilai); i++ {
        fmt.Printf("Masukkan nilai siswa ke-%d: ", i+1)
        fmt.Scan(&nilai[i])
    }

    // Asumsikan nilai awal sebagai acuan
    maks = nilai[0]
    min = nilai[0]

    // Proses mencari maks, min, dan total

            maks = n

        if n < min {
            min = n

    // Hitung rata-rata

    fmt.Println("\n=== HASIL ANALISIS ===")
    fmt.Println("Nilai tertinggi =", maks)

  • SOAL 5: Statistik Data

    Buatlah program untuk menghitung statistik dari array berisi 8 bilangan:

    • Nilai maksimum
    • Nilai minimum
    • Jumlah total
    • Rata-rata
    • Jumlah bilangan di atas rata-rata
    • Jumlah bilangan di bawah rata-rata

Solusi SOAL 5: Program Statistik Data

package main

import "fmt"

func main() {
    var data [8]int
    var maks, min, total int
    var rata float64
    var diAtas, diBawah int

    fmt.Println("=== Program Statistik Data ===")
    fmt.Println("Masukkan 8 bilangan:\n")

    // Input data
    for i := 0; i < len(data); i++ {
        fmt.Printf("Data ke-%d: ", i+1)
        fmt.Scan(&data[i])
    }

    // Inisialisasi
    maks = data[0]
    min = data[0]
    total = 0

    // Hitung maks, min, total
    for _, d := range data {
        if d > maks {
            maks = d
        }
        if d < min {
            min = d
        }
        total += d
    }

    // Hitung rata-rata
    rata = float64(total) / float64(len(data))

    // Hitung data di atas dan di bawah rata-rata
    for _, d := range data {
        if float64(d) > rata {
            diAtas++
        } else if float64(d) < rata {
            diBawah++
        }
    }

    // Tampilkan hasil
    fmt.Println("\n=== HASIL STATISTIK ===")
    fmt.Println("Data:", data)
    fmt.Println("Nilai maksimum:", maks)
    fmt.Println("Nilai minimum:", min)
    fmt.Println("Total:", total)
    fmt.Printf("Rata-rata: %.2f\n", rata)
    fmt.Println("Bilangan di atas rata-rata:", diAtas)
    fmt.Println("Bilangan di bawah rata-rata:", diBawah)
}

💡 Poin Penting tentang Array di Go

  • ✅ Ukuran tetap: Ukuran array tidak bisa diubah setelah deklarasi
  • ✅ Zero value: Elemen yang tidak diinisialisasi akan bernilai 0 (int), "" (string), false (bool)
  • ✅ Pass by value: Array di-copy saat dikirim ke fungsi (beda dengan C)
  • ✅ len(): Fungsi built-in untuk mendapatkan panjang array
  • ✅ Range loop: Cara idiomatik untuk iterasi array di Go
  • ⚠️ Tipe berbeda: [3]int dan [4]int adalah tipe yang berbeda
  • 💡 Untuk ukuran dinamis: Gunakan Slice (akan dibahas di materi selanjutnya)

🔄 Perbandingan Array: Go vs C

Aspek C Go
Deklarasi int arr[5]; var arr [5]int
Panjang sizeof(arr)/sizeof(arr[0]) len(arr)
Pass ke Function By reference (otomatis) By value (harus pakai pointer jika ingin ubah)
Perulangan for(i=0; i<5; i++) for i, v := range arr
Bounds Checking Tidak ada (bisa overflow) Ada (panic jika out of range)
