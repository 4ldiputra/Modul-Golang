1. Pengenalan Golang

1.1 Definisi dan Sejarah

Apa Itu Golang?

Go (atau Golang) adalah bahasa pemrograman open-source yang dikembangkan oleh Google pada tahun 2007 dan dirilis ke publik pada tahun 2009. Diciptakan oleh Robert Griesemer, Rob Pike, dan Ken Thompson, Go dirancang untuk mengatasi masalah dalam pengembangan software skala besar.

Keunggulan Go:

  • Kompilasi cepat dan performa tinggi
  • Concurrency yang mudah dengan goroutines
  • Garbage collection otomatis
  • Sintaks yang sederhana dan mudah dipelajari
  • Standard library yang lengkap
  • Cross-platform compilation

1.2 Instalasi Golang

Instalasi di Windows

  1. Kunjungi situs resmi https://go.dev/dl/.
  2. Download installer Go untuk Windows (file .msi).
  3. Jalankan installer dan ikuti instruksi hingga selesai.
  4. Setelah selesai, buka Command Prompt lalu ketik:
    go version
    Jika muncul versi Go, berarti instalasi berhasil.

Instalasi di macOS dengan Homebrew

  1. Pastikan Homebrew sudah terpasang di sistem. Jika belum, install dengan perintah:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install Go menggunakan brew:
    brew install go
  3. Cek instalasi dengan perintah:
    go version
    Jika muncul versi Go, berarti instalasi berhasil.

2.3 Input dan Output

Dalam Go, untuk melakukan input/output biasanya menggunakan package fmt.

package main import "fmt" func main() { var name string var age int fmt.Print("Masukkan nama: ") fmt.Scanln(&name) fmt.Print("Masukkan umur: ") fmt.Scanln(&age) fmt.Println("Halo,", name, "umur Anda", age, "tahun") }

Penjelasan: - fmt.Print() → menampilkan teks tanpa pindah baris. - fmt.Println() → menampilkan teks dengan pindah baris. - fmt.Scanln(&variabel) → membaca input dari user dan menyimpannya ke variabel.

Latihan Praktek

  • SOAL 1:Buatlah program yang menggunakan inputan dan mendeklarasikan variabel untuk data mahasiswa (nama, NIM, IPK, status aktif)
  • SOAL 2: Buat program yang menyimpan data barang di toko (nama barang, harga, jumlah stok, status tersedia), lalu cetak hasilnya.

Contoh Penyelesaian

package main import "fmt" func main() { var nama string var nim string var ipk float32 var aktif bool fmt.Print("Masukkan Nama: ") fmt.Scanln(&nama) fmt.Print("Masukkan NIM: ") fmt.Scanln(&nim) fmt.Print("Masukkan IPK: ") fmt.Scanln(&ipk) fmt.Print("Apakah masih aktif (true/false): ") fmt.Scanln(&aktif) fmt.Println("\n--- Data Mahasiswa ---") fmt.Println("Nama :", nama) fmt.Println("NIM :", nim) fmt.Println("IPK :", ipk) fmt.Println("Status :", aktif) }

Sistem Manajemen Perpustakaan

Tugas: Buatlah sistem manajemen perpustakaan menggunakan Go dengan fitur CRUD lengkap.

Template Program

package main import ( "fmt" "strings" ) type Book struct { ID int Title string Author string Year int Available bool } type Library struct { Books []Book NextID int } func NewLibrary() *Library { return &Library{ Books: []Book{}, NextID: 1, } } func (l *Library) AddBook(title, author string, year int) { book := Book{ ID: l.NextID, Title: title, Author: author, Year: year, Available: true, } l.Books = append(l.Books, book) l.NextID++ fmt.Println("Buku berhasil ditambahkan!") } func (l *Library) DisplayBooks() { if len(l.Books) == 0 { fmt.Println("Perpustakaan kosong") return } fmt.Println("\n=== DAFTAR BUKU ===") for _, book := range l.Books { status := "Tersedia" if !book.Available { status = "Dipinjam" } fmt.Printf("ID: %d | %s - %s (%d) [%s]\n", book.ID, book.Title, book.Author, book.Year, status) } } func (l *Library) SearchBook(keyword string) []Book { var results []Book keyword = strings.ToLower(keyword) for _, book := range l.Books { if strings.Contains(strings.ToLower(book.Title), keyword) || strings.Contains(strings.ToLower(book.Author), keyword) { results = append(results, book) } } return results } func (l *Library) BorrowBook(id int) { for i := range l.Books { if l.Books[i].ID == id { if l.Books[i].Available { l.Books[i].Available = false fmt.Println("Buku berhasil dipinjam!") } else { fmt.Println("Buku sedang dipinjam") } return } } fmt.Println("Buku tidak ditemukan") } func (l *Library) ReturnBook(id int) { for i := range l.Books { if l.Books[i].ID == id { if !l.Books[i].Available { l.Books[i].Available = true fmt.Println("Buku berhasil dikembalikan!") } else { fmt.Println("Buku tidak sedang dipinjam") } return } } fmt.Println("Buku tidak ditemukan") } func (l *Library) DeleteBook(id int) { for i, book := range l.Books { if book.ID == id { l.Books = append(l.Books[:i], l.Books[i+1:]...) fmt.Println("Buku berhasil dihapus!") return } } fmt.Println("Buku tidak ditemukan") } func showMenu() { fmt.Println("\n=== SISTEM PERPUSTAKAAN ===") fmt.Println("1. Tambah Buku") fmt.Println("2. Tampilkan Semua Buku") fmt.Println("3. Cari Buku") fmt.Println("4. Pinjam Buku") fmt.Println("5. Kembalikan Buku") fmt.Println("6. Hapus Buku") fmt.Println("0. Keluar") fmt.Print("Pilih menu: ") } func main() { library := NewLibrary() // Data awal library.AddBook("Pemrograman Go", "John Doe", 2020) library.AddBook("Web Development", "Jane Smith", 2021) var choice int for { showMenu() fmt.Scan(&choice) switch choice { case 1: var title, author string var year int fmt.Print("Judul: ") fmt.Scan(&title) fmt.Print("Pengarang: ") fmt.Scan(&author) fmt.Print("Tahun: ") fmt.Scan(&year) library.AddBook(title, author, year) case 2: library.DisplayBooks() case 3: var keyword string fmt.Print("Kata kunci: ") fmt.Scan(&keyword) results := library.SearchBook(keyword) if len(results) == 0 { fmt.Println("Buku tidak ditemukan") } else { for _, book := range results { fmt.Printf("ID: %d | %s - %s\n", book.ID, book.Title, book.Author) } } case 4: var id int fmt.Print("ID Buku: ") fmt.Scan(&id) library.BorrowBook(id) case 5: var id int fmt.Print("ID Buku: ") fmt.Scan(&id) library.ReturnBook(id) case 6: var id int fmt.Print("ID Buku: ") fmt.Scan(&id) library.DeleteBook(id) case 0: fmt.Println("Terima kasih!") return default: fmt.Println("Pilihan tidak valid") } } }
